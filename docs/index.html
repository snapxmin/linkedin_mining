研究流程

  1. 生成 site:linkedin.com/in "OKX" 公开搜索查询
  2. 采集搜索引擎公开索引的标题、摘要与 URL
  3. 导入 CSV 并人工复核字段
  4. 统计大学、职位、地点与学位分布
  5. 导出规范化 CSV 供进一步分析

    本地运行完整系统

    git clone https://github.com/snapxmin/linkedin_mining.git
cd linkedin_mining
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[test]"
cp .env.example .env
uvicorn app.main:app --reload

    启动后访问 http://127.0.0.1:8000/ 使用搜索、CSV 导入、复核与导出功能。

    可选：在 .env 中配置 SERPER_API_KEY 启用真实公开索引搜索。
